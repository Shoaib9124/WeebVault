WebVault: Your Anime & Manga Database


WebVault is a personal project designed to rival MyAnimeList, providing an organized platform for anime and manga enthusiasts to explore and track their collections. Currently, WebVault features a search function to help users find their favorite anime or manga titles.

Project Overview


WebVault aims to provide a clean and user-friendly interface for anime and manga enthusiasts to search, discover, and track their favorite titles. The site is in its early stages, with future improvements planned to enhance functionality and user experience.


Features:
Search Functionality: Search for anime or manga titles using the search bar.


Future Features (Planned): Integration with React and a database for user accounts, saved lists, reviews, ratings, and more.


Files in the Repository


HTML Files: Contains the basic structure of the website.


CSS Files: Used to style the site with a responsive design.


JavaScript Files: Contains the logic for the search functionality.
